2025.12.18 虚树

2025/12/9 算法 算法

为了应对如上情况，我们需要严肃学习虚树。

前置知识

  • 树的基本概念、树的存储、树的遍历；
  • 最近公共祖先（LCA）；
  • 树形动态规划、树链剖分、树上倍增等树上算法。

引入

我们先看一道题目：

给定一棵 $n$ 个点的树，边有边权，树上有 $k$ 个关键点（都不为 $1$ 号点），你可以删除一些边，使得 $1$ 号点不能经过剩余的边到达任意一个关键点，最小化你删除的边的边权之和，并输出这个最小值。$1\le m\le n\le10^6$。

树形 $\text{dp}$。设 $dp_x$ 为删除一些边，使得 $x$ 号点不能经过剩余的边到达 $x$ 的子树内（除 $x$ 以外）任意一个关键点的最小代价。即，仅考虑 $x$ 的子树的答案。

遍历一遍整棵树。对于每个点 $u$，枚举它的儿子 $v$，设 $w$ 为 $u$ 到 $v$ 的边的边权：

  • 若 $v$ 不是关键点：$dp_u\to dp_u+\min{dp_v,w}$；
  • 若 $v$ 是关键点：$dp_u\to dp_u+w$。

时间复杂度 $\mathcal O(n)$。

优化

如果多次询问呢？

P2495 【模板】虚树 / [SDOI2011] 消耗战

在一场战争中，战场由 $n$ 个岛屿和 $n-1$ 个桥梁组成，保证每两个岛屿间有且仅有一条路径可达。现在，我军已经侦查到敌军的总部在编号为 $1$ 的岛屿，而且他们已经没有足够多的能源维系战斗，我军胜利在望。已知在其他 $k$ 个岛屿上有丰富能源，为了防止敌军获取能源，我军的任务是炸毁一些桥梁，使得敌军不能到达任何能源丰富的岛屿。由于不同桥梁的材质和结构不同，所以炸毁不同的桥梁有不同的代价，我军希望在满足目标的同时使得总代价最小。

侦查部门还发现，敌军有一台神秘机器。即使我军切断所有能源之后，他们也可以用那台机器。机器产生的效果不仅仅会修复所有我军炸毁的桥梁，而且会重新随机资源分布（但可以保证的是，资源不会分布到 $1$ 号岛屿上）。不过侦查部门还发现了这台机器只能够使用 $m$ 次，所以我们只需要把每次任务完成即可。

对于 $100%$ 的数据，$2\leq n \leq 2.5\times 10^5, 1\leq m\leq 5\times 10^5, \sum k_i \leq 5\times 10^5, 1\leq k_i< n, h_i\neq 1, 1\leq u,v\leq n, 1\leq w\leq 10^5$ 。

如果按照之前的做法，每次询问都要重新 $\text{dp}$，时间复杂度 $\mathcal O(qn)$，正常情况下无法通过。

考虑如何优化这个 $\text{dp}$。

  1. 注意到如果一个子树内没有任何一个关键点，那么直接删去这个子树后对答案不影响。

  2. 考虑剩下的树中的一条链。设上端为 $u$，下端为 $v$，若这条链上除了 $v$ 都只有一个儿子（即不存在“分支”），那么如果我们把中间的点全部去掉，再给 $u,v$ 之间连一条边，边权为原来的边的边权的最小值，这对答案不影响。

结合两点，发现如果一个点不是关键点、且不是某两个关键点的最近公共祖先，那么这个点是可以被删去的。

注意到，经过这样的“删点”，剩下的点最多只有 $2k-1$ 个。

证明

考虑 $\text{lca}(x,y)$，若有 $z$ 在 $\text{lca}(x,y)$ 的子树内，那么 $\text{lca}(x,y)$ 一定是 $\text{lca}(x,z)$ 或 $\text{lca}(y,z)$。
所以只要先按 $\text{dfn}$ 排序，那么任意两个关键点的 $\text{lca}$ 都是某两个编号相邻的关键点的 $\text{lca}$，最多有 $k-1$ 个不同的。再加上原本的 $k$ 个点，故最多只有 $2k-1$ 个。

如果在这样的树上做之前的 $\text{dp}$，总时间复杂度就是 $\mathcal O(\sum k)$ 的，可以通过。

这就是 虚树。

那如何建树？

建树

第一种方法

不妨加入点 $1$。

先按 $\text{dfn}$ 排序，算出所有编号相邻的点的 $\text{lca}$，加上原来的点，得到序列 $a$，序列 $a$ 就是虚树上的所有点。

然后再把序列 $a$ 按 $\text{dfn}$ 排序并去重，连接所有 $(\text{lca}(a_i,a_{i+1}),a_{i+1})$ 就是我们需要构建的虚树了。

想一想，为什么连接所有 $(\text{lca}(a_i,a_{i+1}),a_{i+1})$ 就行？

  • $a_i$ 是 $a_{i+1}$ 的祖先，$\text{dfn}$ 保证了它们之间的路径上没有任何一个点在虚树上，所以要连接 $(a_i,a_{i+1})$，而 $a_i=\text{lca}(a_i,a_{i+1})$；
  • $a_i$ 不是 $a_{i+1}$ 的祖先，$\text{dfn}$ 保证了 $\text{lca}(a_i,a_{i+1})$ 到 $a_{i+1}$ 之间的路径上没有任何一个点在虚树上，所以要连接 $(\text{lca}(a_i,a_{i+1}),a_{i+1})$。

又由于 $a_1=1$ 没有父亲，其余 $a_i$ 都有一个父亲，这样的连边方式正好可以做到不重不漏。

时间复杂度 $O(k\log n)$。

Code

void buildVT(){
    sort(a + 1, a + m + 1, cmp);
    int tmp = m;
    for (int i = 2; i <= tmp; i++){
        a[++m] = lca(a[i - 1], a[i]);
    }
    a[++m] = 1;
    sort(a + 1, a + m + 1, cmp);
    m = unique(a + 1, a + m + 1) - a - 1;
    for (int i = 2; i <= m; i++){
        g[lca(a[i - 1], a[i])].push_back(a[i]);
    }
    return;
}

第二种方法

用一个栈来维护虚树上的一条链。也就是栈里相邻的两个节点在虚树上也是相邻的，而且栈从底部到顶部的 $\text{dfn}$ 是单调递增的，说白了就是某个节点的父亲就是栈中它下面的那个节点。

首先在栈中添加点 $1$。

然后接下来按照 $\text{dfn}$ 从小到大枚举关键点。

  • 若当前的节点与栈顶节点的 $\text{lca}$ 就是栈顶节点的话，则说明它们是在一条链上的。所以直接把当前节点入栈就行了。

与原树的不同之处

一般来说，虚树题的难点就在这，但既然这题是模板，就显然不会太难。

边权要变成原来路径上所有边的最小值。

可以用重链剖分+序列维护区间最值算法（如线段树、ST 表）或者树上倍增等方法处理路径最小值。

然后你就会这道题了。

清空的时候要注意，不能 memset 或者枚举 $n$，这样你的虚树就没有用了。正确做法是：用了什么就清什么。

完整代码

#include <bits/stdc++.h>
using namespace std;
#define FILE(x) freopen(x".in", "r", stdin);freopen(x".out", "w", stdout);
int n, q, m, a[500001];
bool imp[250001];
long long value[250001], dp[250001];
int fa[250001], depth[250001], son[250001], sz[250001], head[250001], dfn[250001], ddfn[250001], tot;
vector <pair <int, long long> > e[250001];
vector <int> g[250001];
int hi[500001];
long long mi[19], ST[250001][19];
void dfs1(int x, int father, int dep){
    fa[x] = father;
    depth[x] = dep;
    sz[x] = 1;
    for (pair <int, long long> u : e[x]){
        if (u.first == father){
            continue;
        }
        value[u.first] = u.second;
        dfs1(u.first, x, dep + 1);
        sz[x] += sz[u.first];
        if (sz[u.first] > sz[son[x]]){
            son[x] = u.first;
        }
    }
    return;
}
void dfs2(int x, int he){
    dfn[x] = ++tot;
    ddfn[tot] = x;
    head[x] = he;
    if (son[x]){
        dfs2(son[x], he);
        for (pair <int, long long> u : e[x]){
            if (u.first == fa[x] || u.first == son[x]){
                continue;
            }
            dfs2(u.first, u.first);
        }
    }
    return;
}
int lca(int u, int v){
    while (head[u] != head[v]){
        if (depth[head[u]] < depth[head[v]]){
            swap(u, v);
        }
        u = fa[head[u]];
    }
    if (depth[u] > depth[v]){
        swap(u, v);
    }
    return u;
}
long long qmin(int l, int r){
    return min(ST[l][hi[r - l + 1]], ST[r - (1 << hi[r - l + 1]) + 1][hi[r - l + 1]]);
}
long long querymin(int u, int v){
    long long res = 1e18;
    while (head[u] != head[v]){
        if (depth[head[u]] < depth[head[v]]){
            swap(u, v);
        }
        res = min(res, qmin(dfn[head[u]], dfn[u]));
        u = fa[head[u]];
    }
    if (u == v){
        return res;
    }
    if (depth[u] > depth[v]){
        swap(u, v);
    }
    res = min(res, qmin(dfn[u] + 1, dfn[v]));
    return res;
}
bool cmp(int x, int y){
    return dfn[x] < dfn[y];
}
void buildVT(){
    sort(a + 1, a + m + 1, cmp);
    int tmp = m;
    for (int i = 2; i <= tmp; i++){
        a[++m] = lca(a[i - 1], a[i]);
    }
    a[++m] = 1;
    sort(a + 1, a + m + 1, cmp);
    m = unique(a + 1, a + m + 1) - a - 1;
    for (int i = 2; i <= m; i++){
        g[lca(a[i - 1], a[i])].push_back(a[i]);
    }
    return;
}
void calc(int x){
    dp[x] = 0;
    for (int u : g[x]){
        calc(u);
        if (imp[u]){
            dp[x] += querymin(x, u);
        } else {
            dp[x] += min(dp[u], querymin(x, u));
        }
    }
    return;
}
int main(){
    ios::sync_with_stdio(false);
    cin.tie(0);
    cout.tie(0);
    int u, v;
    long long w;
    cin >> n;
    for (int i = 1; i < n; i++){
        cin >> u >> v >> w;
        e[u].push_back({v, w});
        e[v].push_back({u, w});
    }
    dfs1(1, 0, 1);
    dfs2(1, 1);
    mi[0] = 1;
    hi[1] = 0;
    for (int i = 1; i <= 18; i++){
        mi[i] = mi[i - 1] << 1;
        for (int j = mi[i - 1]; j < mi[i]; j++){
            hi[j] = i - 1;
        }
    }
    for (int i = 1; i <= n; i++){
        ST[i][0] = value[ddfn[i]];
    }
    for (int j = 1; j <= 18; j++){
        for (int i = 1; i <= n; i++){
            ST[i][j] = min(ST[i][j - 1], ST[i + mi[j - 1]][j - 1]);
        }
    }
    cin >> q;
    while (q--){
        cin >> m;
        for (int i = 1; i <= m; i++){
            cin >> a[i];
            imp[a[i]] = true;
        }
        buildVT();
        calc(1);
        cout << dp[1] << "\n";
        for (int i = 1; i <= m; i++){
            imp[a[i]] = false;
        }
        for (int i = 2; i <= m; i++){
            g[lca(a[i - 1], a[i])].clear();
        }
    }
    return 0;
}
